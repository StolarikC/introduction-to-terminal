"Mastering" the Terminal

The terminal allows us to directly communicate with the computer/device.


KEY ADVANTAGES OF USING THE TERMINAL

  • SPEED!

    Practice makes perfect! Some concepts may seem pointless or feel frustrating, but once you are familiar with the terminal, it can be MUCH faster than using a GUI.

  • ACCESS!

    The terminal provides a 'mainline' into the heart of our computer. Giving us access to areas we normally don't interact with.

  • TOOLS!

    Many of the tools we need are installed and used via the command line. We don't have much of a choice here.

A BRIEF GLOSSARY

  • TERMINAL -- A text-based interface to your computer. Originally a physical object but now we use software terminals.

  • SHELL -- The actual program running on the terminal.

    "BASH" and "Z-SHELL" are two of the most popular shells today.

BASIC COMMANDS

  • "ls" stands for list. The terminal will list the current contents of the directory you are in at that moment.

  • the "~" is a shortcut to your "HOME' directory.

  • "pwd" stands for 'Print Working Directory.' Prints the path to the working directory (where you currently are).

  • "cd" stands for 'Change Directory.'

    IN THE TERMINAL: type "cd *existing directory name*"

    IN THE TERMINAL: type "cd .." to return up one level. THE SPACE IS IMPORTANT!

    You can access ANY 'folder'/'directory' by using "cd /abc/123/xyz" This is known as "ABSOLUTE PATHS" vs. "RELATIVE PATHS" which do not use slashes and 'move' you relative to the current directory in use.

  • "mkdir" stands for "make directory." It will create a new directory or directories.

    Type "mkdir .." to make a new directory, one folder 'above' the current one.

    Type "mkdir Frogs/TreeFrogs SugarGliders," as an example, to create two new directories inside of the pre-existing /frogs. All inside of /pets

  • "man" stands for manual and is followed by another command, which "manual" will provide a description of. This includes a list of parameters called "flags" that can be used with the given command.

    TYPE "Q" TO EXIT

    Type "ls -l" to access the flag "l", whose function does not matter for now.

  • "touch" creates a new file (or multiple) from the command line.

    Type "touch fileName.txt otherFile.html" or whatever file type (.html, .js, .css, etc.) you want to create. List them as shown to create multiple in a single command.

    IF called upon an existing file, it will "touch" it, and update its "last modified" time.

  • "rm" will PERMANENTLY "remove" the called file.

    There is also "rmdir" to remove a directory, but ONLY works if the directory called is empty.

    "rm -rf" stands for "remove -recursive force", meaning all items inside the called directory (recursive) will be deleted without asking permissions (force).
